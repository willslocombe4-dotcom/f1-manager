# Brainstorm Mode

Set the F1 Director to **INACTIVE** mode for free brainstorming.

## Instructions

1. Read `.opencode/context/f1-director-context.md`
2. Change `**Mode:** ACTIVE` to `**Mode:** INACTIVE`
3. Save the file
4. Respond with:

```
Brainstorm Mode activated

Director is dormant. You can now chat freely with @f1-idea-designer.
Any approved designs will be saved to the backlog.

Commands:
- /activate - Switch to build mode
- /backlog - Show saved ideas and activate
- /status - Check current mode
```
