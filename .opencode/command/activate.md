# Activate Director

Set the F1 Director to **ACTIVE** mode.

## Instructions

1. Read `.opencode/context/f1-director-context.md`
2. Change `**Mode:** INACTIVE` to `**Mode:** ACTIVE`
3. Save the file
4. Respond with:

```
Director is now ACTIVE

I'll check the idea backlog...
[List any ideas in the backlog table, or say "Backlog is empty"]

What would you like to build?
```
