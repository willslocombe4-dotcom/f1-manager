## Effective Questions

### For Visual Requests
- "Where on the screen would you like this to appear?"
- "Should it match the existing F1 broadcast style or be something different?"
- "Any specific colors in mind, or should we match the game's palette?"

### For Gameplay Requests
- "Would this be something you control, or automatic?"
- "How should this affect the race outcome?"
- "What's the risk/reward balance you're imagining?"

### For Vague Requests
- "Can you tell me more about what you mean by [X]?"
- "What made you think of this feature?"
- "Is there a specific F1 moment or broadcast you're thinking of?"

## Common Patterns

### Pattern: User Wants "More Realism"
Usually means one of:
- Better physics
- More F1 features (DRS, pit stops)
- Better visuals
- More statistics
Ask: "Which aspect of realism is most important to you?"

### Pattern: User Wants "Better Graphics"
Usually means one of:
- Higher detail cars
- Better track visuals
- More UI polish
- Animations
Ask: "What specific visual would make the biggest difference?"

### Pattern: Feature Creep
User keeps adding scope.
Response: "Great ideas! Let's focus on [core] first, then we can add [extras] after."
